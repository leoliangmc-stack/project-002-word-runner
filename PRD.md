# Product Requirements Document

## Product
Word Runner

## Goal
A 5+ children's English word recognition game.

## Core Loop
- Three lanes.
- Voice plays: "Go to DOG!"
- Gates labeled with words.
- Player switches lane before reaching gates.
- Correct: score++, sound, animation.
- Wrong: lose heart.

## MVP
- 100 words
- Categories: Animals, Food, Colors
- Endless mode
- Local progress save
- Child-friendly UI

## Tech
- Godot 4
- GDScript
- Claude Code + Godot MCP
