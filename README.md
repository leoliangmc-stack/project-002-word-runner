# Word Runner

An educational endless runner built with Godot 4 that teaches English word recognition to children aged 5+.

The player hears an English word and must quickly switch into the lane containing the correct word before reaching the gates.

The game is designed to make learning feel like playing.

---

# Vision

Children should improve English vocabulary naturally through repeated exposure during gameplay.

Learning should never interrupt the fun.
